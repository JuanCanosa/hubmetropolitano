P1 · Mais fácil · Zero código
1. Crie uma planilha com aba "Dashboard"
2. Colunas: A=métrica · B=valor · C=histórico(JSON)
3. Métricas: parceiros, mrr, assinantes, nps, ocup
4. Arquivo → Compartilhar → "qualquer pessoa com o link"
5. Cole o ID (parte da URL entre /d/ e /edit)

Supabase → Eventos · Vagas · Parceiros · Obras
P2 · Banco real · Realtime · Free tier — supabase.com
Tabelas necessárias (crie no Supabase → Table Editor):
hub_kpis — metric(text), value(text), history(jsonb)
hub_eventos — nome, data, local, categoria, cor, entrada, publico, descricao
hub_vagas — nome, empresa, local, salario, tipo, categoria, icone, descricao
hub_parceiros — nome, categoria, score, icone, diferencial, ativo(bool)
hub_obras — nome, categoria, local, status, progresso, previsao, investimento
Row Level Security: habilite com policy SELECT anon em todas as tabelas
